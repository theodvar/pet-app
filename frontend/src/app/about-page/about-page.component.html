My Pet Application

Σύνοψη Εφαρμογής

Το My Pet App είναι μια εφαρμογή που αφορά τα κατοικίδια μας. Σκοπό έχει οι ιδιοκτήτες τους να μοιραστούν να μοιραστούν χρήσιμες δημιουργικές πληροφορίες με άλλους ιδιοκτήτες. Πιο συγκεκριμένα ο κάθε χρήστης της εφαρμογής μπορεί να δει καταχωρύσεις άλλων ιδιοκτητών σχετικά με τις παρακάτω κατηγορίες:

  • Σημεία ενδιαφέροντος για κατοικίδια (κτηνίατροι, pet-shops, pet-friendly μέρη)
  • Προτεινόμενες βόλτες για σκύλους
  • Πάρκα σκύλων

Αλλά επίσης ο χρήστης μπορεί να καταχωρήσει και τις δικές του προτάσεις σχετικά με τις παραπάνω κατηγορίες. Η εφαρμογή συνδέει όλα τα παραπάνω με τον χώρο. Καθώς τόσο για να περιηγηθείς στις κατηγορίες όσο και για να καταχωρήσεις κάτι σε αυτή γίνεται μέσω χάρτη. Κάθε μια από τις κατηγορίες υποστηρίζει μια γεωμετρία τα σημεία ενδιαφέροντος για κατοικίδια εμφανίζονται ή καταχωρούνται ως σημεία. Οι βόλτες για σκύλους ως γραμμές και τα πάρκα σκύλων ως πολύγωνα.

Χαρακτηριστικά Εφαρμογής

Η εφαρμογή είναι ένα rest API γιατί έχει τα εξής χαρακτηριστικά:

  • Client-Server αρχιτεκτονική
  • Resource-based API
  • Χρήση HTTP μεθόδων (GET, POST, PUT, DELETE)
  • Ανταλλαγή δεδομένων σε JSON & GeoJSON

Πιο συγκεκριμένα η εφαρμογή περιέχει 15 endpoints, 5 για την κάθε κατηγορία petpointofinterest, dogwalks, dorparks. Εκτελώντας βασικές CRUD πράξεις. Το My Pet App αποτελείται από 2 panel, το δεξιά περιέχει το χάρτη και το αριστερά αποτελείται από 2 dropdown menu και κουμπιά. Το 1ο dropdown menu μπορεί να εμφανίσει τις καταχωρημένες στη βάση προτάσεις επιλέγοντας την κάθε κατηγορία. Ο χρήστης μπορεί έτσι να περιηγηθεί πάνω στο χάρτη και πατώντας πάνω σε κάθε στοιχείο να εμφανιστούν με popup οι καταχωρημένες πληροφορίες για αυτό. Μέσω του popup μπορεί κάποιος να επεξεργαστεί τις πληροφορίες του αντικειμένου και στο χάρτη την γεωμετρία του. Ο χάρτης ανανεώνεται αυτόματα. Επίσης μπορεί και να διαγράψει το αντικείμενο. Μετά τη διαγραφή προτείνεται να πατηθεί το κουμπί refresh map ώστε να επαναφορτωθούν ο χάρτης και τα ανανεωμένα αντικείμενα για όλες τις κατηγορίες. Στο 2ο dropdown menu μπορεί ο χρήστης να προσθέσει τη δική του πρόταση για κάθε μια από τις 3 κατηγορίες επιλέγοντας την. Σχεδιάζει τη γεωμετρία στο χάρτη και μετά εμφανίζεται η φόρμα για τη συμπλήρωση των αντίστοιχων πληροφοριών. Το pan πάνω στο χάρτη λειτουργεί με πατημένο το δεξί κλικ. Μπορεί ο χρήστης να κάνει zoom και να σχεδιάσει το σημείο, τη γραμμή ή το πολύγωνο που θέλει. Η ολοκλήρωση της σχεδίασης γραμμής γίνεται πατώντας κλικ ξανά πάνω στο τελευταίο σημείο, ενώ στο πολύγωνο η ολοκλήρωση υλοποιείται πατώντας πάνω στο αρχικό σημείο ώστε να κλείσει. Για τη γραμμή ή το πολύγωνο υπάρχει στο αριστερό panel και undo button μπορεί να αφαιρέσει τον τελευταίο σχεδιασμένο κόμβο σε περίπτωση λάθους. Για να βγει ο χρήστης από το mode της σχεδίασης μπορεί να επιλέξει στο 2ο dropdown menu την επιλογή none. Μετά τη σχεδίαση προτείνεται να πατηθεί το κουμπί refresh map ώστε να επαναφορτωθούν ο χάρτης και τα ανανεωμένα αντικείμενα για όλες τις κατηγορίες.

Τεχνολογίες

  • Backend: Python με Flask
  • Swagger: Τα endpoints τεκμηριώνονται με Swagger,127.0.0.1:5000/swagger/
  • Frontend: Angular & OpenLayers
  • Βάση δεδομένων: PostgreSQL

Κίνητρο & Προκλήσεις

Πέρα από το ενδιαφέρον μου για τον προγραμματισμό και την επιστήμη υπολογιστών, η συγκεκριμένη εφαρμογή αποτελεί την τελική μου εργασία στα πλαίσια του Bootcamp Coding Factory από το ΚΕΒΙΔΙΜ του ΟΠΑ. Οι προκλήσεις κατά την ανάπτυξης της εφαρμογής ήταν πολλές, κυρίως όμως λόγω του γεωχωρικού χαρακτήρα των δεδομένων και του δυναμικού χαρακτήρα που πρέπει να έχουν πάνω στο χάρτη. Αρκετές ξεπεράστηκαν άλλες παραμένουν ώστε να εισαχθούν σε μελλοντική βελτιωμένη έκδοση.
